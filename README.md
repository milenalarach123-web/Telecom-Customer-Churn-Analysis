# Telecom Customer Churn Analysis
Proyecto de análisis de factores de cancelación de clientes en una empresa de telecomunicaciones utilizando Power BI y Power Query.

Objetivo: Analizar los factores asociados a la cancelación de clientes en una empresa de telecomunicaciones.

## Tecnologías
- Power BI
- Power Query
- DAX

## Hallazgos
- Los contratos mensuales presentan mayor riesgo de cancelación.
- Los clientes sin soporte técnico muestran mayor churn.
- Los usuarios de fibra óptica presentan mayores tasas de abandono.
